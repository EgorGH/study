1. За один проход входных данных находим:
	x_maxp - макс. значение по модулю из всех х > 0 на оси у
	x_minp - мин. значение по модулю из всех х > 0 на оси у
  	x_maxn - макс. значение по модулю из всех х < 0 на оси у
  	x_minn - мин. значение по модулю из всех х < 0 на оси у
  	y_maxp - макс. значение по модулю из всех у, у которых x > 0
  	y_maxn - макс. значение по модулю из всех у, у которых x < 0

2. Выводим:
	max(|(x_maxp - x_minp) * y_maxp / 2|, |(x_maxn - x_minn) * y_maxn / 2|)
